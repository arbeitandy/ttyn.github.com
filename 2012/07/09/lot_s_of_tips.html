  Ctrl-x Ctrl-e actually editing the command line

  • get public ip 
      curl http://ip4.me 2>/dev/null | sed -e 's#<[^>]*>##g' | grep '^[0-9]'

      clear && vlock -c
  • real path 
      pwd -P

text process | 文本處理

  • print a line in file 
      $ sed -n 4p $file_name 
  -n switch turns off default output

  $ sed -n -e 3,60p -e 1p $file_name
