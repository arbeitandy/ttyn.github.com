在centos 6.0 上安装tmux

  1. add RPMforge yum repo (64bit)
    - rpm —import http://apt.sw.be/RPM-GPG-KEY.dag.txt
    - wget http://packages.sw.be/rpmforge-release/rpmforge-release-0.5.2-2.el6.rf.×86_64.rpm
    - rpm -i rpmforge-release-0.5.2-2.el6.rf.*.rpm
  2. install prerequisite for tmux
    - yum install -y libevent
  3. install tmux
    - yum —enablerepo rpmforge install -y tmux
